DANS Archive example

We have several examples of Transactional Web Archives installed. The DANS ( http://www.dans.knaw.nl/ ) has TWA archive distributed between TWA at http://theresourcedepot.org/000010/timegate/{anyuri} and wayback installation (http://lanlproto.santafe.edu:8080/memento/timegate/{anyuri}) populated from data unloaded from TWA in WARC file format. The TimegateDirector installed at http://theresourcedepot.org/000010D/{anyuri}. Let check that all components of Memento Framework in place:

  • DANS Content server has Link header, which points to TimegateDirector. 
    •  
curl -D my.txt -I http://www.dans.knaw.nl/
HTTP/1.1 200 OK
Date: Thu, 24 May 2012 16:56:29 GMT
Server: Apache/2.2.3 (Red Hat)
Link: <http://theresourcedepot.org/000010D/http://www.dans.knaw.nl/>;rel=timegate
X-Powered-By: PHP/5.1.6
Set-Cookie: SESS4a0091108fb271e05f34da7cf77c975f=j7bc90c1hte9k9sarpkts3h092; expires=Sat, 16 Jun 2012 20:29:49 GMT; path=/
Expires: Sun, 19 Nov 1978 05:00:00 GMT
Last-Modified: Thu, 24 May 2012 16:56:29 GMT
Cache-Control: store, no-cache, must-revalidate
Cache-Control: post-check=0, pre-check=0
Connection: close
Content-Type: text/html; charset=utf-8
  • TimegateDirector redirects to appropriate archive: 
    •  
curl -D my.txt -I http://theresourcedepot.org/000010D/http://www.dans.knaw.nl
HTTP/1.1 302 Moved Temporarily
Date: Thu, 24 May 2012 17:19:34 GMT
Location: http://www.theresourcedepot.com/000010/timegate/http://www.dans.knaw.nl

Connection: close
Content-Type: text/plain; charset=UTF-8
Content-Language: nl

    Note discovery link to index timemap

LANL Archive example


 We are archiving several web servers : http://mementoweb.org/ see 
 for more detals,
http://lanlsource.lanl.gov/hello   see more info here,
and http://www.shared-canvas.org/ . All mentioned websites get archived to the Transactional Web Archive at  
http://mementoarchive.lanl.gov. As described in  Memento Framework Discovery Specifications  we have host-meta document at http://mementoarchive.lanl.gov/.well-known/host-meta supporting TimeGate and TimeMap discovery.





End-User Interfaces to  Archive


 
To get smooth "Time Travel" experience download the MementoFox plugin.
To install the plugin, drag and drop this XPI file over any open Firefox window, which will invoke the Add-on installation window.
After closing and restarting the FireFox go to Tools ->Add-ons->Extentions to check that you have MementoFox extension listed.
