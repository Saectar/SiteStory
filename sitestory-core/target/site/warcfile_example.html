WARC file

We are using request,response and metadata WARC types. In metadata section we convey additional information about memento response, such as period when digest of the page bitestream was the same (start, end),number of hits during that period. As you can see WARC-Concurrent-To: parameter used to connect response record with corresponding metadata record.

WARC-Type: warcinfo
WARC-Date: 2012-05-04T17:50:32Z
WARC-Filename: MEM-20120504175032789-00000-~~.warc.gz
WARC-Record-ID: <urn:uuid:6c9458de-88ff-4042-8bac-3f7088259637>
